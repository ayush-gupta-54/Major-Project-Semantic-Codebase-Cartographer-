# Semantic Codebase Cartographer

A Streamlit-based tool to analyze and visualize Python code structure using graph-based representations.

## 🚀 Features
- Parses Python codebase
- Builds dependency graph
- Visualizes relationships using interactive graphs
- Helps understand large projects easily

## 🛠 Tech Stack
- Python
- Streamlit
- NetworkX
- PyVis
- PyTorch (if used)
- NumPy
